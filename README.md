# SmartPath-Kids
Scams inflicted HK$8.1 billion in losses across 43,212 cases in Hong Kong in 2025, despite a slight 2.9% drop in cases—online investment fraud alone surged 30.7% to HK$3.58 billion, averaging HK$700,000 per victim. With youth increasingly targeted via digital channels, there’s urgent demand for scalable scam-prevention education.
